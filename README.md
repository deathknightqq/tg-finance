# FinBot

Telegram-бот финансового учёта поверх PDF-выписок Kaspi. Личный проект на двух пользователей.

Полная спека: [docs/finbot_spec.md](docs/finbot_spec.md) — сборка идёт строго по чанкам из раздела 7, с чекпоинтами.

## Стек

Python 3.12, aiogram 3.x (long polling), pdfplumber, SQLite + SQLAlchemy 2.x, pytest, Docker.

## Настройка

1. Скопировать `.env.example` в `.env` и вписать токен бота.
2. `.env`, базы данных и любые PDF-выписки в git не коммитятся (см. `.gitignore`).
